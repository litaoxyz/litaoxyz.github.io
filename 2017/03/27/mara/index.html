2017厦门马拉松

发表于 2017-03-27 更新于 2017-07-12 分类于 运动

　 厦马作为我2017第一个马拉松，继秦皇岛，野鸭湖，郑开，武汉，北京，长沙之后，人生的第七个马拉松。我热爱这项赛事，参加马拉松的过程，虽然每次都无比痛苦无聊，但是每次都坚持下来了，厦马后半程天气比较热，和郑开一样，都晒伤，但是都完赛了，跑马开始慢慢来,觉得挺舒服的，后来就比较无聊了，除了首马和别人一起跑的，后来的都是自己跑的。一个人的马拉松，我从中感受到了不可言状的奇妙感觉，马拉松可以磨砺我的韧性，降低了我对幸福感的阀值,即使是很简单的事情我也可以从中感受到幸福。对其一切事物的看法或追求都变得很低很低，感觉这个世界是多么的柔软。马拉松让我明白自己的极限,让我明白了坚持的意义。这就是目前我对马拉松的理解。
